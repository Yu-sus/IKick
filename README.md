# IKick

IKick is a keyless open-source script designed for *Kick a Lucky Block*. The project focuses on simplicity, readability, and flexibility, allowing users to understand, modify, and expand the codebase easily.

It is intended as both a functional script and a learning resource for those interested in Lua scripting and Roblox script structure.

---

## 📌 Features

- Keyless execution (no key system required)
- Lightweight and easy-to-read code
- Fully editable source
- Simple function-based structure
- Designed for customization and learning

---

## ⚙️ Customization

IKick is built to be modified.

You are free to:
- Change existing functions
- Add new features
- Remove unnecessary systems
- Adapt the script to your own workflow

If you modify the script, it is recommended to set:

```lua
_IKick.Modified = true
```
